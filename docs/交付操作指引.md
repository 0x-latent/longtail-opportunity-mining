# 数据交付指南

本文档指导您如何上传数据文件到指定的云存储空间。

---

## 一、安装上传工具

请下载并安装 **COSBrowser**（腾讯云官方文件管理工具）：

- Windows：https://cos5.cloud.tencent.com/cosbrowser/cosbrowser-setup-latest.exe
- Mac：https://cos5.cloud.tencent.com/cosbrowser/cosbrowser-latest.dmg

安装完成后打开 COSBrowser，使用我们提供的 **SecretId** 和 **SecretKey** 登录。

---

## 二、上传步骤

1. 登录后，进入存储桶 `social-listening-1307163315`
2. 找到**您对应的项目文件夹**（已由我方创建，请勿修改文件夹名称）
3. 将准备好的 CSV 文件**拖拽到该文件夹中**即可完成上传

> 注意：您只有上传权限，无法查看或下载文件夹中已有的文件，这是正常的。

---

## 三、需要交付的文件

每次交付包含 **2 个 CSV 文件**：

| 文件 | 说明 |
|------|------|
| 原始数据 | 帖子/评论的原始文本数据 |
| 标签表 | 标签层级体系 + 对应的原文关键词 |

---

## 四、文件命名规范

```
{项目名}_posts_{日期}.csv      ← 原始数据
{项目名}_labels_{日期}.csv     ← 标签表
```

**示例：**
```
cold_cough_posts_20260401.csv
cold_cough_labels_20260401.csv
```

**规则：**
- 项目名使用英文小写 + 下划线（如 `cold_cough`、`baby_skin`）
- 日期为交付日期，格式 YYYYMMDD
- 两个文件的项目名和日期必须一致
- 文件格式为 CSV，编码 UTF-8

---

## 五、原始数据格式（_posts_）

| 字段名 | 必填 | 说明 |
|--------|------|------|
| id | 否 | 帖子唯一标识，留空则由系统自动生成 |
| platform | 否 | 来源平台（小红书、抖音、微博等） |
| date | 否 | 发布日期，格式 yyyy-MM-dd，未知可留空 |
| title | 否 | 帖子标题，没有标题的平台可留空 |
| text | **是** | 帖子正文，保留原始内容 |

**示例：**

```
id,platform,date,title,text
,小红书,2026-01-10,宝宝反复咳嗽怎么办,三岁宝宝反复咳嗽有痰快一个月了，吃了头孢和氨溴索都不见好，有没有宝妈有经验的 #宝妈日常
,抖音,2026-02-03,,晚上咳嗽咳到睡不着，喝了蜂蜜水稍微好一点，但是第二天又开始了，崩溃
,微博,2026-01-28,支原体肺炎住院记录,孩子支原体肺炎住院第五天，从阿奇换成了多西环素，今天终于不烧了
```

**注意事项：**
- 帖子正文保留原始内容，**不需要**手动清除 URL、话题标签（#xxx#）、@提及、表情符号等
- 如果原始数据列名不是上述英文名（如用"文本"、"平台"），请在交付时备注列名对照

## 六、标签表格式（_labels_）

只需要两列：

| 字段名 | 必填 | 说明 |
|--------|------|------|
| label_path | **是** | 标签层级路径，用 `/` 分隔，层数不限 |
| keywords | **是** | 该标签在原文中对应的关键词，用英文分号 `;` 分隔 |

### 层级路径说明

```
第一层/第二层/第三层/.../最后一层
```

- **第一层**为维度名（如"症状"、"场景"、"方案"），建议 4-8 个维度
- **中间层**为分类/子分类，层数自由
- **最后一层**为具体标签名

### 示例

```
label_path,keywords
症状/咽喉/咳嗽,咳嗽;一直咳;干咳;痰咳;咳个不停;咳得厉害
症状/咽喉/喉咙痛,喉咙痛;嗓子痛;咽痛;喉咙不舒服;嗓子发炎;嗓子疼
症状/鼻部/流鼻涕,流鼻涕;鼻涕;鼻塞;鼻子不通气;鼻涕很多
症状/全身/发烧,发烧;发热;低烧;高烧;体温高;烧到39度;反复发烧
场景/时间/夜间,晚上;夜里;半夜;睡前;夜间;凌晨
场景/人物/幼儿,小孩;孩子;宝宝;幼儿;娃;崽崽;小朋友
场景/地点/幼儿园,幼儿园;上学;入园;园里;班上
方案/西药/美林,美林;布洛芬;布洛芬混悬液
方案/西药/头孢,头孢;头孢克肟;头孢克洛
方案/中医/推拿,小儿推拿;推拿;按摩;捏脊
方案/食疗/蜂蜜水,蜂蜜水;蜂蜜柠檬;蜂蜜
效果评价/正面/有效,有用;管用;见效快;效果好;好多了;终于好了
效果评价/负面/无效,没用;不管用;没效果;白吃了;一点用没有
情绪/焦虑,焦虑;担心;害怕;慌;急死了;愁死了
情绪/崩溃,崩溃;难受;顶不住;受不了;太难了
```

### 关键词填写要求

1. **必须是帖子原文中实际出现的词语**，不是标签的规范名称
   - 例：标签是"发烧"，关键词应包含用户常说的"烧到39度"、"反复发烧"等
2. **同义词、口语化表达都要列出**
   - 例："喉咙痛"还应包含"嗓子疼"、"咽痛"
3. **长短搭配**：既要有短词（"咳嗽"）也要有短语（"咳个不停"）
4. 关键词之间用**英文分号** `;` 分隔（不是中文分号 `；`）
5. 每个标签建议 **3-10 个**关键词，覆盖最常见的表述即可
6. 同一维度下的不同标签，**关键词尽量不要重叠**

## 七、常见问题

**Q: 上传后看不到文件？**
A: 正常，您的账号只有上传权限。上传成功后会有提示，我们会在后台处理。

**Q: 文件上传失败？**
A: 请检查：(1) 网络是否正常 (2) SecretId/SecretKey 是否填写正确 (3) 是否上传到了正确的文件夹

**Q: 数据有更新怎么办？**
A: 用新的日期命名即可（如 `cold_cough_posts_20260415.csv`），系统会自动使用最新日期的文件。

**Q: 可以上传非 CSV 文件吗？**
A: 目前只支持 CSV 格式。如果原始数据是 Excel，请先另存为 CSV（UTF-8 编码）。
